## 📖 About

**WalX Images** serves as the image repository for the **WalX** project.

This repository is intended to be consumed as a Git submodule by the main WalX repository, allowing the gallery to remain lightweight while keeping the wallpaper collection independent and easy to update.

---

## ✨ Features

* 📷 Thousands of wallpapers
* 🗂️ Organized into categories
* 🚀 Optimized for Hugo GalleryDeluxe
* 🔄 Easy updates through Git submodules
* 💾 Original image quality preserved whenever possible

## Usage

This repository is designed to be mounted into the main WalX repository as a Git submodule.

```bash
git submodule add https://github.com/roxxadiiii/walx-images.git content/images
```
